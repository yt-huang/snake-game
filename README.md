# Snake Game 🐍

A sleek, dark-themed Snake game built with pure HTML, CSS, and JavaScript.

- **Canvas-based rendering** with smooth pixel-art style
- **Keyboard controls** — Arrow keys to steer, Space to restart
- **Responsive** — works on desktop and mobile
- **Zero dependencies** — just open `index.html` in a browser

## Play

👉 https://ying62506799.github.io/snake-game/

## Local

```bash
# Option 1: Open directly
open index.html

# Option 2: Python server
python3 -m http.server 8888
```
